Project structure of: james4ever0/prometheous

  • prometheous Root directory containing all system components and user data
    • document_agi_computer_control Document-based AI computer control system
      • build_website.py Website generator from code directory using argparse and template files.
      • cache_db_context.py Cache management with TinyDB and file operations.
      • code_view_demo.py Repeated greeting function with name input.
      • codepiece_summarizer.py Comments to summary using LLM
      • custom_doc_writer.py Custom document writer with progress visualization and queue-based comment processing.
      • ensure_fdignore_with_doc.py Ensures "docs" in .fdignore file.
      • estimate_utils.py Estimates processing time for file list.
      • example.py Hello world example, print function.
      • identify_utils.py Determine language of file, randomize if "text".
      • llm.py OpenAI LLM initialization with debugging, chain execution, and GPT-4 options.
      • path_select_utils.py Interactive path selection for docs and caching.
      • recursive_document_writer.py Recursive, Jinja2, JSON, Python indexing document webpages.
      • sitemap_generator
        • main.py Sitemap generator script
      • slice_utils.py Slice dictionaries with specified size.
      • stdout_redirect_progress Asynchronous stdout progress tracking with Python.
        • main.py Subprocess asynchronous execution with progress bar.
        • main_once.py Python async stdout redirect, progress bar tracking
        • main_once_char_by_char.py Progress bar text app with asyncio and stdout/stderr handling
        • test.py Custom print function with progress bar and sleep.
        • test_no_patch.py Simulates progress bar with "hello world", adjustable sleep time.
      • title_generator
        • main.py Generates file titles from metadata and comments using AI
      • tree_markdown_view_folder_hierarchy Generates cacheable Markdown directory hierarchy.
        • high_level.py Update briefs for file summaries and directories.
        • main.py Python script generates cacheable Markdown-formatted directory hierarchy.
        • main_recursive.py Generates efficient Markdown tree with HTML styling
      • vectorstore_embedding_chat_rag Document AI chat, vector store, control, embedding.
        • docarray_test.py DocArray tests Ollama embeddings and HnswIndex.
        • llamaindex_test.py Test LLaMa Index with Vector Store
        • llamaindex_vector.py Constructs VectorStoreIndex from documents using service context
        • test.py LlamaEmbeddings for querying, document embedding.
        • test_hnswlib.py Performs k-NN search on HNSW index with random embeddings
      • visual_file_selector_by_ignore_rules Visual file selector by ignore rules, tree utilities.
        • async_utils.py Asynchronous file utilities: read, run command.
        • display_tree_structure.py Visual file selector by ignore rules. Displays tree structures with stats.
        • file_copy_by_fd.py Python script copies files using fd and argparse.
        • main.py Visual file selector with ignore rules and progress bar.
        • main_pyjom.py Visualize file selector using ignore rules for efficient file control.
        • run_simple.py Python script compares file trees with ignore rules, creates JSON output.
        • test_tree_visual.py Visual file selector with ignore rules.
        • tree_utils.py Tree utility functions for visual file selectors.
    • search_filepath_by_name
      • search_filepath.sh Search files in current directory using fzf.
